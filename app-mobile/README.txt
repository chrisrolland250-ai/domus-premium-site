Domus Premium - Application Mobile Web (avec icône)

Fichiers :
- index.html : application web mobile (saisie chantiers, devis, suivi clients, stats, export CSV).
- apple-touch-icon.png : icône utilisée pour le raccourci sur l'écran d'accueil (iOS) et comme favicon.

Installation sur GitHub Pages :
1. Crée un dossier (par exemple 'app-mobile') dans ton dépôt GitHub de ton site.
2. Copie index.html et apple-touch-icon.png dans ce dossier.
3. Pousse les modifications sur GitHub.
4. L'application sera accessible via une URL du type :
   https://domuspremium.fr/app-mobile/

Sur iPhone :
- Ouvre l'URL dans Safari.
- Menu "Partager" > "Ajouter à l'écran d'accueil".
- iOS utilisera automatiquement apple-touch-icon.png comme icône du raccourci.
