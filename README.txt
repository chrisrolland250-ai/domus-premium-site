# Section "À propos" — Domus Premium
Dernière mise à jour : 12/09/2025

## Fichiers
- `apropos_section.html` : section HTML prête à coller dans `index.html` (après le hero).
- `apropos_styles.css` : CSS minimal pour la mise en page (à copier dans votre feuille principale).
- `nav_snippet.html` : élément de menu à ajouter (lien vers `#apropos`).
- `optional_organization.jsonld` : balisage JSON-LD Organization (optionnel).

## Intégration (rapide)
1. Ajoutez le lien de menu (`nav_snippet.html`) dans votre navigation.
2. Collez `apropos_section.html` dans `index.html`.
3. Ajoutez `apropos_styles.css` à votre CSS global.
4. Remplacez l’image `assets/about/portrait-travail.jpg` par votre photo (1200×800, WebP conseillé).
5. (Optionnel) Collez le JSON-LD dans le `<head>`.
